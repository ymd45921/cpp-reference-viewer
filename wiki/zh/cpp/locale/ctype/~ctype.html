在标头 <locale> 定义
protected: ~ctype();

析构 std::ctype 刻面。此析构函数为受保护且为虚函数（由于基类析构函数为虚函数）。std::ctype 类型的对象，同大多数刻面一样，只能在最后一个实装此刻面或用户定义的派生自 std::ctype 并实现公开析构函数的刻面的 std::locale 离开作用域时，才会被销毁。

示例

运行此代码
#include <iostream>
#include <locale>
 
struct Destructible_ctype : public std::ctype<wchar_t>
{
    Destructible_ctype(std::size_t refs = 0) : ctype(refs) {}
    // 注意：隐式析构函数为公开
};
 
int main()
{
    Destructible_ctype dc;
    // std::ctype<wchar_t> c; // 编译错误：受保护析构函数
}
