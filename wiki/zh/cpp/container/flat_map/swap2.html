在标头 <flat_map> 定义
friend void swap( std::flat_map& lhs, std::flat_map& rhs ) noexcept;
 (C++23 起)

为 std::flat_map 特化了 std::swap 算法。交换 lhs 与 rhs 的内容。调用 lhs.swap(rhs)。


参数

lhs, rhs - 要交换内容的容器

返回值

（无）

复杂度

与交换各底层容器相同。

示例

运行此代码
#include <algorithm>
#include <iostream>
#include <flat_map>
 
int main()
{
    std::flat_map<int, char> alice{{1, 'a'}, {2, 'b'}, {3, 'c'}};
    std::flat_map<int, char> bob{{7, 'Z'}, {8, 'Y'}, {9, 'X'}, {10, 'W'}};
 
    auto print = [](std::pair<const int, char>& n)
    {
        std::cout << ' ' << n.first << ':' << n.second;
    };
 
    // 打印交换前的状态
    std::cout << "Alice:";
    std::for_each(alice.begin(), alice.end(), print);
    std::cout << "\nBobby:";
    std::for_each(bob.begin(), bob.end(), print);
    std::cout << '\n';
 
    std::cout << "-- SWAP\n";
    std::swap(alice, bob);
 
    // 打印交换后的状态
    std::cout << "Alice:";
    std::for_each(alice.begin(), alice.end(), print);
    std::cout << "\nBobby:";
    std::for_each(bob.begin(), bob.end(), print);
    std::cout << '\n';
}

输出： 

Alice: 1:a 2:b 3:c
Bobby: 7:Z 8:Y 9:X 10:W
-- SWAP
Alice: 7:Z 8:Y 9:X 10:W
Bobby: 1:a 2:b 3:c
