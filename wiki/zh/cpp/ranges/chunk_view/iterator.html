template< bool Const >
class /*iterator*/
 (C++23 起)
(仅用于阐述*)

当底层视图 V 为 common_range 时，是 chunk_view::begin 和 chunk_view::end 的返回类型。

仅在 V 实现 forward_range 时给出。

const 限定的重载返回 /*iterator*/<true> 类型，非 cosnt 限定的重载返回 /*iterator*/<false> 类型。

成员类型

成员类型 定义
Parent (私有) 如果 Const 等于 true 则为 const chunk_view 否则为 chunk_view。
(仅用于阐述的成员类型*)
Base (私有) 如果 Const 等于 true 则为 const V 否则为 V。
(仅用于阐述的成员类型*)
iterator_category std::input_iterator_tag
iterator_concept
  • 如果 Base 实现 random_access_range，则为 std::random_access_iterator_tag，
  • 如果 Base 实现 bidirectional_range，则为 std::bidirectional_iterator_tag，
  • 否则为 std::forward_iterator_tag。
value_type decltype(views::take(ranges::subrange(current_, end_), n_));。
difference_type ranges::range_difference_t<Base>

数据成员

成员名称 定义
current_ (私有) ranges::iterator_t<Base>，持有指向底层序列中当前块的首个元素的迭代器(也就是一个块的首部)。
(仅用于阐述的成员对象*)
end_ (私有) ranges::sentinel_t<Base>，持有指向底层序列尾部的哨位。
(仅用于阐述的成员对象*)
n_ (私有) ranges::range_difference_t<Base>，持有区块中元素的初始数量（最后一块可以较小。
(仅用于阐述的成员对象*)
missing_ (私有) ranges::range_difference_t<Base>，通常等于 ranges::advance(current_, n_, end_);。
(仅用于阐述的成员对象*)
